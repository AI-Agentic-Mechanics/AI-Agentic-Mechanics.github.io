# Outbound Message Pack (Updated: ROI First, Safety Built In, Team Ownership)

Use this structure for each prospect:
- Personalize one line with a real trigger (new location, hiring, acquisition, SLA pressure, service backlog).
- Lead with one measurable KPI and a fixed-scope pilot.
- Position safety guardrails as built-in, not an optional add-on.
- Frame team training as phase 2 after pilot success.

Core offer sequence:
1. Phase 1: 2-4 week workflow pilot with one KPI.
2. Built in: safety guardrails (permissions, approvals, logging, data controls).
3. Phase 2: team enablement and handoff for internal ownership.
